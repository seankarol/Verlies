# Verlies

Early version of a couch co-op roguelite dungeon crawler.

Built in [Godot v4.3](https://godotengine.org/download).

## Controls

### Keyboard & Mouse

- WADS to move
- Left mouse to attack
- Space to join
- Escape to pause

### Gamepad

- Left joystick to move
- R2 to attack
- Start to join/pause
